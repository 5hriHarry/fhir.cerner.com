Search

Search for Patients that meet supplied query parameters:

GET /Patient?:parameters

Parameters

Name Type Description
_id token The logical resource id associated with the resource.
birthdate date The patientâ€™s date of birth. Example: 1961-01-16
identifier token A patient identifier. Example: 01022228
name string A portion of either family or given name of the patient. Example: Peters
telecom token The value in any kind of telecom details of the patient. Example: (816) 475-2374
